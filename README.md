 When the user runs the program, they should first run the Price Calculator code.
The user will be prompted to input a “yarder size” of either 20 or 30
Once the information is input the code will calculate the price and then open it in
a "total_cost.txt" file. Download this file onto your PC and save it for later use.
After completing the Price calculator then, user should run the Franchise Fee
code. User will need to input a specific city*.
Once the information is input the code will calculate the franchise fee and then
open it in a “franchise_fee.txt” file. Download this file onto your PC and save it
for later use.
Finally, the user will run the Call log code. User will be asked to input info* that
will essentially be added to a final receipt.
Before running this code user must open both the "total_cost.txt" and the
“franchise_fee.txt” for the values to be added onto the receipt.txt file.
*Yarder size range: 20/30
*Franchise fee cities: Coral Gables, Doral, Miami Beach, Miami Gardens, North Miami,
Sweetwater.
*The only Existing users saved to the code have names “Tony” and “Antonio”
